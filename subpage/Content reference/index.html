關於Open Source Physics

        隨著科技的演進，電腦也在教育機構中扮演著不可或缺的角色，許多與實驗和理論相關的物理也都必須仰賴電腦的幫助。然而，即使科技在教育和研究領域進步得如此快速，計算物理學在傳統大學物理中仍然是很重要的一環，計算和物理模型建構在這方面則明顯的不足。儘管學生的生活中到處充斥著各種物理現象的模擬，不論是課堂上教師、教科書的出版商或是電影導演所提供的，但是只有很小一部份的學生能夠真正的體會這些，而 Open Source Physics 則希望能夠彌補這樣的缺失。

        當學生並非積極主動的投入在計算和模型建構時，他們損失了許多可以藉由電腦模擬而學習的東西。雖然物理模型建構的方法能夠不依靠電腦進行，但使用電腦能夠方便學生學習比較困難且耗時學習的事物，並且將結果視覺化、和其他人共同交流。計算物理學以及模型建構能夠讓學生確實的體會到物理模型的建構、描述、解釋和預測一個物理現象。我們相信這兩者的結合能夠讓學生完整的理解物理的理論和實驗。

        Open Source Physics (http://www.compadre.org/osp/) 是一個開源而且豐富的物理資源網站，收集了許多物理相關的模擬程式和網頁。而 Open Source Physics 中文站則是希望能夠盡量把它中文化，使得更多學生能因此更容易的學習物理，並且對相關事物感到更有興趣。
