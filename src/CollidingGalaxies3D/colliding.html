軌道和碰撞星系3D

在軌道和碰撞星系的3-D模型中，軌道的延伸和碰撞星系二維模型分為三個層面。 這樣的模式，也是由 宋靈澤 和 沃爾夫岡·克里斯蒂安 擴展開來的。該模型假設銀河 系的中心的點群眾和星星不互相影響，只是銀河系內核互相與明星個別互動。 星星 在每個星系都以一個平面中心的3D圓形軌道開始。 當兩個星系相互穿越時，他們會產生長螺旋尾巴。

控制

控制軌道和碰撞星系三維模型的是：

  • 開始/暫停 按鈕:  模擬影像播放和暫停
  • 步伐 按鈕: 將模擬影像一次前進一個步驟
  • 時間重置 按鈕: 重置模擬影像回到T=0，但         保持當前的變量值。
  • 復原 按鈕: 講模擬影像重置回原始值
  • x 和 y 位置: 改變星系1和2的初始位置
  • vx 和 vy 的速度向量: 改變星系1和2的初始速度方向
  • m1 和 m2 質量: 改變星系核心的質量（不超越太陽質量為條件）。
  • ring1 和 ring2 環型半徑長度: 改變星系1和2的環狀半徑
  • XY - XZ - YZ 落下: 改變星系1的方向(紅色的)
  • 縮放塊:  滑塊允許使用者放大和縮小。

老鼠

小鼠模型是基於兩個首先由哈勃太空望遠鏡見到碰撞星系。初始條件為採取這兩個星系模仿圖像。 每個星系都有恆星在不同的平面，這意味著在尾部會看起來像一個螺旋而       其他的從其他角度看尾巴會看起來像一個長的直線。 在現實中，直尾也是一個螺旋， 但當我們以直的方式看時，你也一樣看不到。

  • M1 = M2 = 100
  • 環狀1 = 環狀2 = 10
  • X1 = 50 和 X2 = -50
  • Y1 = Y2 = 0
  • Z1 = 20 和 Z2 = 0
  • vX1 = -5 和 Vx2 = 5
  • vY1 =2.5 和 Vy2 = -2.5
  • vZ1 = vZ2 = 0
  • 將銀河1定位在XZ平面
