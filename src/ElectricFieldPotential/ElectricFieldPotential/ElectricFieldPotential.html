電場和電位的模型

這個電場和電位的EJS模型顯示了在圓球形和圓柱形的系統中電場與電位的分布。 在這兩種系統中，都有著呈同心圓的帶電面，並且在帶電面外皆為接地。 這項模型會反映一個測試的點電荷（黃色的）在不同的同心圓面之中所受到的力和電壓。 有關EJS的更多資訊可由此獲得: <http://www.um.es/fem/Ejs/>.

練習:

  1. 書本上常用二維的形式來表現電場的向量。 但請注意，如圖中所示，圓球和圓柱在二維空間中的的圖形是相同的:


    假設你現在有一塊呈薄板狀的帶電導體。  那麼你預測它的電場和電位分布會比較接近 (a) 球體 還是 (b) 圓柱?  請解釋。

  2. 執行模擬動畫，你可以 放大 (按住Shift並點擊拖曳), 旋轉 (點擊拖曳) 或平移 (按住Ctrl並點擊拖曳) 3D視角。  請問在你的觀察中，球體和圓柱系統中電場的作用形式有何不同? 
  3. 請問當你改變平面在 z 軸上的位置時 ( 拉動z軸 ) ，在球體和圓柱系統中"等電位圖" 分別有什麼樣的變化? 請解釋。
  4. 你可以透過點擊"等電位圖", 上的點來獲得該點的電壓。 當平面在 z=0時， 請問球體和圓柱系統中，電壓分別是如何隨著與電荷中心的距離變化的?
  5. 這項模擬動畫也會顯示一個測試電荷 ( 黃色的 ) 所受到的電力之向量， 試著到處移動這顆電荷。 當你移動這顆電荷時，模擬動畫會顯示它現在的位置(單位:公分) 和它所受的力(單位 10-12牛頓)。 如果這顆測試電荷所帶的電量為 1 pC (10-12 庫倫), 請問在這兩個模型中 圓柱模型的線電荷密度 (對於帶電的長直線) 和 球體模型中心的電荷帶電量分別為多少?
