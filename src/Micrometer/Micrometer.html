千分尺模擬

千分尺

這個微米模型展示了操作的原理以及實際微米的物理部分。  因為距離太小以至於無法測量，所以千分尺利用螺釘來擴增距離，使之能夠變為足夠大的範圍 ，且能直接讀取螺桿的轉動。一微米的精確度產生於心臟線程的精確度。千分尺的基本工作原理是：一個製作精確的螺桿，其轉動可直接且與定量的軸向移動為正相關(反之亦然)， 透過被稱為螺桿引領的常數。 一螺桿引領是其向軸向旋轉一整圈且往前移動之距離。 (360°). (在大多數的線程上[也就是，在所有的單啟動線程]，引領和螺距指基本相同的概念。 ) 在螺釘的適當引領與主要直徑上， 在軸向運動的方向上給個定量，在所得的圓周運動中放大。

千分尺架 (橘色)
C形體維持了鐵砧和桶子之間相對於彼此的恆定關係。 因為擴張與收縮會影響測量的結果 ，所以它是厚的，如此一來才能做到最小化的擴張與收縮。這個框架是重的，所以有較高的比熱，防止用手/手指握住時造成大量的升溫， 具有一個0.01的毫米儀器，將其劃分為2圈=100=1.00mm，使其與真正的微米做連結。

鐵砧 (灰色)
閃亮的部分：紡錘向前移動，樣本則不休息。

袖 / 桶 / 股 (黃色)
固定輪的部分與其線性度。有時以游標尺標記。

鎖核 / 鎖環 / 頂針鎖 (藍色)
凸邊部（或控制桿），一個可以上緊到保持主軸固定的，像是用來當暫時保持一個測量

螺釘
(看不見) 千分尺之核心。 它位於桶的內部。

紡錘 (深綠色)
閃亮的圓筒：頂針造成往鐵砧前進的運動。而位於底部面板的綠色滑塊則控制主軸的位置。

頂針 (綠色)
可用一根拇指轉動的部分。 刻度之標記。

棘輪 (藍綠色)
(模擬時不會動)裝置在手柄的末端上，藉由滑動校準扭矩來限制施加的壓力。

千分尺模型有一個物件(黑色)，利用左上角之滑塊，控制此物件在鐵砧與主軸(爪)中y方向之運動。 圖片可做拖拉操作。
而利用左下角的滑塊去控制此物件在鐵砧與主軸(爪)中x方向之運動。

在左邊底部滑塊是零誤差控制，允許  如果在使用千分尺有0.15毫米（最大）或-0.15mm （分鐘）零誤差。

重置鍵可使千分尺模型恢復最初之設定。
